# pipGUI
pip的GUI版本
依赖模块（Python自带）：
    urllib （from request）（用来发送检查更新请求）
    tkinter（from Tk,Button,Label,messagebox,simpledialog,Scrollbar,RIGHT,Y,YView,Menu,OptionMenu,ttk）
依赖模块（2.0才需要）：
   bs4（BeautifulSoup）from BeautifulSoup

功能：
    1、安装模块。
    2、查看已安装模块。
    3、卸载模块。
2.5新更新：
    1、可以改变安装源。
    2、GUI部分更改
    3、判断有无Wifi。
    4、不需要第三方模块BeautifulSoup。
2.6新更新：
    1、可以手动更新。
